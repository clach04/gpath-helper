svg parser 2.1 -paths

This a proof of concept, that parses an svg image and then creates the code needed to render those coordinates using Gpath on the pebble platform

To try it with a custom svg, Open this link in Chrome to draw an svg polygon in your browser, click on the "edit source" button and paste the svg code below in the box labelled: "Paste svg code here," then click "get code"

currently only works with an svg path image composed of straight lines.
